## 概要

このアプリはQRコードでメモを共有するアプリです。
URLにランダムな文字列が生成され、相手とQRコードを渡すことでメモを共有できます。

## デザインカンプ

今回は「adobeXD」で簡易的なデザインカンプを作りました。以下のURLにて未実装部分のデザインも閲覧できます。
